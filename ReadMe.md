# ReadMe

## Environment

It requires *sklearn* and *matplotlib* to run the code.

Use

> pip install matplotlib

and

> pip install scikit-learn

to install.

## See ground truth

Use

> python show_ground_truth.py

to print the true clusters.

## Run clustering algorithms

Try

> python kMeans.py
>
> python DBScan.py
>
> python OPTICS.py
